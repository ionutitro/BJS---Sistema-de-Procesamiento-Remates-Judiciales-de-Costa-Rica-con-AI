# BJS---Sistema-de-Procesamiento-Remates-Judiciales-de-Costa-Rica-con-ayuda-de-la-tecnol-gia-AI
El  BJS es una plataforma integral de automatización y análisis diseñada para la extracción, procesamiento e indexación inteligente de remates judiciales en Costa Rica. El sistema procesa boletines oficiales en formato PDF utilizando Inteligencia Artificial local y heurísticas legales avanzadas.
